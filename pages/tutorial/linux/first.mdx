## What you need?

If you fresh in ubuntu you need to install some packages :

1. Change mirror ubuntu server to main server, [see this](https://askubuntu.com/questions/104695/how-do-i-change-mirrors-in-ubuntu-server-from-regional-to-main)
2. Install [Git](https://git-scm.com/downloads)
3. Install [NVM](https://github.com/nvm-sh/nvm) (Node Version Manager)
4. Install [zsh](https://github.com/ohmyzsh/ohmyzsh/wiki/Installing-ZSH) (Z Shell)
5. Install [Kitty](https://sw.kovidgoyal.net/kitty/) atau [Fig](https://fig.io/) (Terminal Emulator)
6. Install [Oh My Zsh](https://ohmyz.sh/) (Zsh Framework)
7. Install [Powerlevel10k](https://github.com/romkatv/powerlevel10k) (Zsh Theme)
8. Install [Fira Code](https://github.com/tonsky/FiraCode) (Font)
9. Install [Nerd Font](https://www.nerdfonts.com/) (Font)
10. Install [Composer](https://getcomposer.org/download/)
11. Install [php](https://www.php.net/downloads.php)
