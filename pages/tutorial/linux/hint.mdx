## Install package `.deb`

If you have a `.deb` file and want to install it you can choose between two options using terminal or GUI. This my opinion from each other :

### This list how to use `dpkg`

1. Install package

```bash
sudo dpkg -i [name_package.deb]
```

2. View list of installed packages

```bash
dpkg -l
```

3. Verify package

```bash
dpkg --status [package_name]
```

4. Remove package using `dpkg`

```bash
dpkg -l | grep keyword #To find out the exact package name
```

```bash
sudo dpkg -r [package_name] #or sudo dpkg --remove [package_name]
```
